# Footer

The `Footer` is a component that acts as the footer within 
`App`
.
> **Note**: Learn more about using this component [here](
../learning/app-component
).

## Context Variables

This component does not have any context variables.

## Props

## API

## Events

The `Footer` component uses these theme variables to customize its appearance:

- `color-bg`
- `font-size`
- `height`
- `max-content-width`
- `horizontal-padding`
- `vertical-padding`
- `vertical-alignment`
