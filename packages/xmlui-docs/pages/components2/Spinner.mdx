# Spinner

The 
`Spinner`
 component is an animated indicator that represents a particular action in progress
without a deterministic progress value.
While it is visible, the action is yet to be completed; on completion, the UI logic may opt to remove the component.

## Using the `Spinner`

```ueml copy
<HStack padding="$space-1">
  <Spinner />
</HStack>
```

## Context Variables

This component does not have any context variables.

## Props

### `delay`

The delay in milliseconds before the spinner is displayed.

Use the buttons to toggle between the two `Spinners`.

```ueml copy {8-9}
<VStack padding="$space-1" gap="$space-1">
  <var name="noDelay" value="{true}" />
  <var name="yesDelay" value="{false}" />
  <HStack gap="$space-0_5">
    <Button label="No delay" onClick="noDelay = true; yesDelay = false;" />
    <Button label="1000milsec delay" onClick="noDelay = false; yesDelay = true;" />
  </HStack>
  <Spinner when="{noDelay}" delay="0" />
  <Spinner when="{yesDelay}" delay="1000" />
</VStack>
```

### `fullScreen`

If set to `true`, the component will be rendered in a full screen container.
The default value is `false`.

```ueml copy
<HStack padding="$space-1">
  <Spinner fullScreen="true" />
</HStack>
```

## Styling

The `Spinner` has the following theme variables:

- `color-border-Spinner`
- `size-Spinner`
- `thickness-Spinner`

```json copy
"size-Spinner": "3rem",
"thickness-Spinner": "12px",
"color-border-Spinner": "red"
```
