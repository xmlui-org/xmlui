# EmojiSelector

The 
`EmojiSelector`
 component provides users with a graphical interface to browse,
search and select emojis to insert into text fields, messages, or other forms of communication.

## Context Variables

This component does not have any context variables.

## Props

### `autoFocus`

This prop is used to auto focus on the component when the user navigates to the page.

```ueml copy
<HStack padding="0.5rem">
  <EmojiSelector autoFocus="true" />
</HStack>
```

## API

This component does not provide any API.

## Events

### `select`

This event is fired when the user selects an emoticon from this component.

```ueml copy /onSelect="(emoji) => { selected = emoji }"/
<HStack var.selected="" gap="1rem">
  <EmojiSelector onSelect="(emoji) => { selected = emoji }" />
  <Text value="Selected emoji: {selected}" />
</HStack>
```
