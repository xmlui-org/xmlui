# Icon

This component is the representation of an icon.
It is small visual element, which is used to represent certain functions, features, or types of content within a user interface.

## Props

### `fallback`

This optional property provides a way to handle situations when the original `name` does not denote a valid icon in the [icon registry]().

```ueml copy /fallback="trash"/
<Icon name="noicon" fallback="trash" />
```

<Playground
  previewOnly
  name="Example: fallback"
  height={40}
  app={`<Icon name="noicon" fallback="trash" />`}
/>

### `name`

This string property specifies the unique name of the icon to display; it is case-sensitive. The engine looks up the icon in its registry and determines the visual element the component will show.
This property is required so that the `Icon` can show itself. Nothing is displayed if the specified icon is not found in the [icon registry]().

```ueml copy /name/
<HStack>
  <Icon name="message" />
  <Icon name="note" />
  <Icon name="cog" />
  <Icon name="start" />
  <Icon name="some-non-existing-icon" />
  <Icon name="some-non-existing-icon-with fallback" fallback="trash" />
</HStack>
```

### `size`

This property defines the display size of the `Icon`.
Note that setting the `height` and/or the `width` of the component will override this property.

| Value | Description               |
| :---- | :------------------------ |
| `xs`  | Extra small size          |
| `sm`  | Small size                |
| `md`  | Medium size **(default)** |
| `lg`  | Large size                |

```ueml copy /size/
<HStack gap="1rem" >
  <Icon name="like" />
  <Icon name="like" size="xs" />
  <Icon name="like" size="sm" />
  <Icon name="like" size="md" />
  <Icon name="like" size="lg" />
</HStack>
```

## Events

This component does not have any events.

## Styling

The `Icon` component uses these theme variables:

- `size-Icon`
- `thickness-stroke-Icon`:

```json
{
  "id": "custom",
  "name": "Custom Theme",
  "themeVars": {
    "size-Icon": "4rem",
    "thickness-stroke-Icon": "10"
  }
}
```

<Playground
    name="Example: container children"

    height={140}

        id: "custom",

        themeVars: {
          "size-Icon": "4rem",

        },
    }]}
    app={`<Icon name="like" />`}
