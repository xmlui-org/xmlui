# Theme

This component allows defining a particular theming context for its nested components.

When changing the theming context, you can change its tone or even specify individual theme variables to change the appearance of the nested contents.

You can learn more about the `Theme` component in the [Themes and Styles Overview](/learning/themes/overview/#style-scopes) article.

## Using Theme

In contrast to other components, `Theme` allows the use of any property name to define the value of a particular theme variable. The component injects a theme variable using the particular property name. The theme variable will use the property value.

The following sample sets the theme to a dark tone and sets several `ProgressBar`-related theme variables through `Theme` properties:

```ueml copy {2-7}
<Theme 
  tone="dark"
  color-bg-ProgressBar="cyan"
  color-indicator-ProgressBar="purple"
  thickness-ProgressBar="12px"
  radius-indicator-ProgressBar="12px"
  radius-Progressbar="4px">
  <VStack padding="1rem" gap="1rem" backgroundColor="$color-bg-primary">
    <ProgressBar value="0"/>
    <ProgressBar value="0.2"/>
    <ProgressBar value="0.6"/>
    <ProgressBar value="1.0"/>
  </VStack>
</Theme>
```

## Props

### `tone`

This property allows switching the current theme's tone to another one, for example, from light to dark or vice versa.

```ueml copy {2,8,9,15}
<Fragment>
  <Theme tone="dark">
    <VStack padding="1rem" gap="1rem" backgroundColor="$color-bg-primary" >
      <H3>Use dark tone:</H3>
      <ProgressBar value="0"/>
      <ProgressBar value="0.6"/>
    </VStack>
  </Theme>
  <Theme tone="light">
    <VStack padding="1rem" gap="1rem" backgroundColor="$color-bg-primary">
      <H3>Use light tone:</H3>
      <ProgressBar value="0"/>
      <ProgressBar value="0.6"/>
    </VStack>
  </Theme>
</Fragment>
```

### `themeId`

This property allows switching to another theme using the specified theme identifier.

```ueml copy {2,8,9,15}
<Fragment>
  <Theme themId="xmlui">
    <VStack padding="1rem" gap="1rem" backgroundColor="$color-bg-primary" >
      <H3>Use tabler theme:</H3>
      <ProgressBar value="0"/>
      <ProgressBar value="0.6"/>
    </VStack>
  </Theme>
  <Theme themeId="base-light">
    <VStack padding="1rem" gap="1rem" backgroundColor="$color-bg-primary">
      <H3>Use light tone:</H3>
      <ProgressBar value="0"/>
      <ProgressBar value="0.6"/>
    </VStack>
  </Theme>
</Fragment>
```

## Events

This component does not have any events.

## Styling

The `Theme` component is a styling wrapper that influences nested components' visual appearance. It cannot be styled.
