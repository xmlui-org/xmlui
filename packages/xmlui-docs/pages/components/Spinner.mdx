# Spinner

A `Spinner` component is an animated indicator that represents a particular action in progress without a deterministic progress value.
While it is visible, the action is yet to be completed; on completion, the UI logic may opt to remove the component.

## Using the `Spinner`

```ueml
<HStack padding="1rem">
    <Spinner />
</HStack>
```
