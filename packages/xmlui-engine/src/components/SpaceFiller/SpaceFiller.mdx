## Using `SpaceFiller`

### In `Stack`

The `SpaceFiller` in a `Stack` component pushes the components after the `SpaceFiller` to the other end of the container.

```ueml copy
<HStack>
  <Stack width="64" height="64" backgroundColor="red" />
  <SpaceFiller />
  <Stack width="64" height="64" backgroundColor="blue" />
</HStack>
```

### In `FlowLayout`

The `SpaceFiller` in a `FlowLayout` component acts as a line break for a row. All components after it start from a new line.

```ueml copy
<FlowLayout>
  <Stack width="20%" height="30" backgroundColor="red" />
  <SpaceFiller />
  <Stack width="20%" height="30" backgroundColor="blue" />
</FlowLayout>
```
