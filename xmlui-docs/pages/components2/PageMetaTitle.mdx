# PageMetaTitle

A `PageMetaTitle` component allows setting up (or changing) the app title to display with the 
current browser tab.

## Context Variables

This component does not have any context variables.

## Props

### `value`

This property sets the page's title to display in the browser tab.

## API

This component does not provide any API.

## Events

This component does not have any events.

## Styling

This component does not have any styles.
