# VStack [#component-vstack]

This component is inherited from <SmartLink href="/components/Stack">Stack</SmartLink>

See also: <SmartLink href="/components/CHStack">CHStack</SmartLink>, <SmartLink href="/components/CVStack">CVStack</SmartLink>, <SmartLink href="/components/HStack">HStack</SmartLink>

`VStack` is a specialized, shorthand version for the regular [Stack](./Stack) component with a vertical orientation.
See also: [HStack](./HStack.mdx), [CHStack](./CHStack.mdx), [CVStack](./CVStack.mdx).

```ueml copy
<VStack gap="1rem">
  <Stack height="32px" width="32px" backgroundColor="red" />
  <Stack height="32px" width="32px" backgroundColor="blue" />
  <Stack height="32px" width="32px" backgroundColor="green" />
</VStack>
```

> **Note**: You cannot change the orientation of `VStack` from vertical to horizontal by setting the `orientation` prop, as the engine ignores that setting.
