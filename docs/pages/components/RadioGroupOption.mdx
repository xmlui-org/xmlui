# RadioGroupOption [#component-radiogroupoption]

A single radio button within a radio button group

## Properties

### `enabled`

This boolean property value indicates whether the component responds to user events (`true`) or not (`false`).

Default value: `true`.

### `label`

The option's label

### `value`

The value of the option
