# RadioGroupOption

The `RadioGroupOption` component defines a single option that can be used within a [`RadioGroup`](./RadioGroup).

## Properties

### `enabled`

This property indicates whether the component accepts user inputs (`true`) or not (`false`). The default value is `true`.


```ueml copy {2-3}
<RadioGroup initialValue="first">
  <RadioGroupOption value="first" enabled="false" />
  <RadioGroupOption value="second" enabled="true" />
</RadioGroup>
```

### `label`

Provides a label for a particular option. If not provided, the control falls back to display the value as the label.

```ueml copy /label="Custom Label"/
<RadioGroup>
  <RadioGroupOption value="first" label="Custom Label" />
  <RadioGroupOption value="no-label" />
</RadioGroup>
```

<Playground
  previewOnly
  name="Example: label"
  height={50}
  app={`

      <RadioGroupOption value="first" label="Custom Label" />
      <RadioGroupOption value="no-label" />
    </RadioGroup>
  `}
/>

### `value`

Denotes the actual value that can be selected in a [`RadioGroup`](./RadioGroup.mdx) by this component.

```ueml copy /value="first"/
<RadioGroup>
  <RadioGroupOption value="first" />
</RadioGroup>
```

## Events

This component does not have any events.
