This component renders an HTML `track` tag.



## Properties

### `default`

Specifies that the track is to be enabled if no other track is more suitable

### `kind`

Specifies the kind of text track

### `label`

Specifies the title of the text track

### `src`

Specifies the URL of the track file

### `srcLang`

Specifies the language of the track text data
