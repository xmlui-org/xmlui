# EmojiSelector

<Callout type="info" emoji="ℹ️">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

The `EmojiSelector` component provides users with a graphical interface to browse, search and select emojis to insert into text fields, messages, or other forms of communication.

## Properties

### `autoFocus`

This property sets the label of the component.

Default value: `false`.

```ueml copy
<HStack padding="0.5rem">
  <EmojiSelector autoFocus="true" />
</HStack>
```

## Events

### `select`

This event is fired when the user selects an emoticon from this component.

```ueml copy /onSelect="(emoji) => { selected = emoji }"/
<HStack var.selected="" gap="1rem">
  <EmojiSelector onSelect="(emoji) => { selected = emoji }" />
  <Text value="Selected emoji: {selected}" />
</HStack>
```
