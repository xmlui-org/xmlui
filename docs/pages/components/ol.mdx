This component renders an HTML `ol` tag.



## Properties

### `reversed`

Specifies that the list order should be descending (9, 8, 7, ...)

### `start`

Specifies the start value of the first list item

### `type`

Specifies the kind of marker to use in the list (e.g., '1', 'A', 'a', 'I', 'i')
