# HSplitter

`HSplitter` is a specialized, shorthand version for the regular [Splitter](./Splitter) component with a vertical orientation.
See also: [VSplitter](./VSplitter.mdx).

```ueml copy
<HSplitter height="100%">
  <Stack backgroundColor="lightblue" height="100%" />
  <Stack backgroundColor="darksalmon" height="100%" />
</HSplitter>
```

> **Note**: You cannot change the orientation of `HSplitter` from vertical to horizontal by setting the `orientation` prop, as the engine ignores that setting.
