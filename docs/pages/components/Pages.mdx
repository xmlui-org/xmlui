# Pages

`Pages` is a placeholder within `App` to define the contents of particular navigation targets.

> **Note**: You can learn more details about using this component [here](../learning/app-component).

## Properties

### `defaultRoute`

This property specifies the default route the app navigates to at its startup.

## Events

This component does not provide event handlers to any system or user event.

## Styling

This component is a placeholder; it does not support styling.
