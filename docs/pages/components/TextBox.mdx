# TextBox

`TextBox` is a component that allows users to input and edit text data.

The `TextBox` is an input control component and is often used in forms.
See the [Using Forms](../learning/forms.mdx) guide for details.

## Properties

### `autoFocus`

If this boolean prop is set to true, the `TextBox` input will be focused when appearing on the UI.

### `validationStatus`

This prop is used to visually indicate status changes reacting to form field validation.

| Value     | Description                                           |
| :-------- | :---------------------------------------------------- |
| `valid`   | Visual indicator for an input that is accepted        |
| `warning` | Visual indicator for an input that produced a warning |
| `error`   | Visual indicator for an input that produced an error  |

```ueml copy
<VStack gap="0.5rem">
    <TextBox />
    <TextBox validationStatus="valid" />
    <TextBox validationStatus="warning" />
    <TextBox validationStatus="error" />
</VStack>
```

## API

### `value`

You can query this read-only API property to get the input component's current value.

See an example in the `setValue` API method.

### `setValue`

You can use this method to set the component's current value programmatically.

```ueml copy {3, 9, 12}
<VStack gap="0.5rem">
    <TextBox
      id="textbox"
      readOnly="true"
    />
    <HStack gap="0.5rem">
      <Button 
        label="Add Text" 
        onClick="textbox.setValue('This is an example')" />
      <Button 
        label="Remove Text" 
        onClick="textbox.setValue('')" />
    </HStack>
</VStack>
```

## Events

### `didChange`

This event is triggered after the user has changed the field value.

Write in the input field and see how the `Text` underneath it is updated in parallel.

```ueml copy {2}
<VStack gap="0.5rem" var.field="">
    <TextBox initialValue="{field}" onDidChange="(val) => field = val" />
    <Text value="{field}" />
</VStack>
```

<Playground
  previewOnly
  name="Example: didChange"
  height={80}
  app={`<VStack gap="0.5rem" var.field="">
    <TextBox initialValue="{field}" onDidChange="(val) => field = val" />
    <Text value="{field}" />
</VStack>`}
/>

### `gotFocus`

This event is triggered when the `TextBox` receives focus.
Clicking on the `TextBox` in the example demo changes the label text.
Note how clicking elsewhere resets the text to the original.

```ueml copy {3-4}
<TextBox
    initialValue="{focused === true ? 'I got focused!' : 'I lost focus...'}"
    onGotFocus="focused = true"
    onLostFocus="focused = false"
    var.focused="{false}"
/>
```

<Playground

## Styling

The `TextBox` component uses these theme variables to customize its appearance:

- `radius-TextBox`
- `color-text-TextBox`
- `color-bg-TextBox--disabled`
- `thickness-border-TextBox`
- `style-border-TextBox`
- `color-border-TextBox--disabled`
- `color-bg-TextBox--disabled`
- `color-text-TextBox--disabled`
- `color-border-TextBox-default`
- `color-border-TextBox-error`
- `color-border-TextBox-warning`
- `color-border-TextBox-success`
- `color-adornment-TextBox`
- `color-placeholder-TextBox`

```json
"radius-TextBox": "30px",
"color-adornment-TextBox": "red",
"thickness-border-TextBox": "4px"
```

<Playground
  name="Example: styling"
  previewOnly
  height={60}
  defaultTheme="custom"
  themes={[
    {
      id: "custom",
      name: "Custom Theme",
      themeVars: {
        "radius-TextBox": "30px",
        "color-adornment-TextBox": "red",
        "thickness-border-TextBox": "4px",
      },
    },
  ]}
  app={`<TextBox startIcon="hyperlink" />`}
/>

To style all input controls not just this component, use `Input` instead of the `TextBox` segment:

```
color-border-Input: "#0033FF"
```

`Input` affects the following controls:

- [`Checkbox`](./Checkbox.mdx)
- [`NumberBox`](./NumberBox.mdx)
- [`RadioGroupOption`](./RadioGroupOption.mdx)
- [`Select`](./Select.mdx)
- [`Switch`](./Switch.mdx)
- [`TextArea`](./TextArea.mdx)

See [custom themes](../learning/themes/overview.mdx) for styling details.
