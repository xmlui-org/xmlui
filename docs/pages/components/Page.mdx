# Page

<Callout type="info" emoji="ℹ️">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

The `Page` component defines what content is displayed when the user navigates to a particular URL that is associated with the page.

## Properties

### `url`

The URL of the route associated with the content.
