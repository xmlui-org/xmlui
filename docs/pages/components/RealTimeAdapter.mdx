# RealTimeAdapter

<Callout type="info" emoji="ℹ️">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

`RealTimeAdapter` is a non-visual component that listens to real-time events through long-polling.

## Properties

### `url`

This property specifies the URL to use for long-polling.

## Events

### `eventArrived`

This event is raised when data arrives from the backend using long-polling.

## Context Values

This component does not have any context values.

## Exposed Members

This component does not expose any members.

## Styling

This component does not have any theme variables.
