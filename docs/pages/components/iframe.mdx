# iframe [#component-iframe]

<Callout type="warning" emoji="ðŸ“”">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

This component renders an HTML `iframe` tag.



## Properties

### `allow`

Specifies a feature policy for the iframe

### `allowFullScreen`

Specifies that the iframe can be displayed in full-screen mode

### `height`

Specifies the height of the iframe

### `loading`

Specifies the loading behavior of the iframe

### `name`

Specifies the name of the iframe

### `referrerPolicy`

Specifies the referrer policy for the iframe

### `sandbox`

Specifies a set of extra restrictions for the content in the iframe

### `src`

Specifies the URL of the page to embed

### `srcDoc`

Specifies the HTML content of the page to embed

### `width`

Specifies the width of the iframe
