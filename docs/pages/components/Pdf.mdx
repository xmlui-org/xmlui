# Pdf

The `Pdf` component provides a read-only preview of a particular PDF document's contents.

## Properties

### `src`

This property defines the source URL of the PDF document stream to display.

<Playground
  name="Example: using Pdf"
  previewOnly={true}
  height={160}
  app={`
    <VStack>
      <Pdf src="https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" />
    </VStack>
  `}
/>

## Events

This component does not have any events.

## Styling

The `Pdf` component does not support styling.
