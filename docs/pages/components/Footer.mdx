# Footer 

The `Footer` is a uniquely named component that acts as the footer within the `App` component.

> **Note**: Learn more about using this component [here](../learning/app-component).

## Properties

This component does not have any properties.

## Events

This component does not have any events.

## Styling

The `Footer` component uses these theme variables to customize its appearance:

- `color-bg`
- `font-size`
- `height`
- `max-content-width`
- `horizontal-padding`
- `vertical-padding`
- `vertical-alignment`
