# NumberBox

A `NumberBox` component allows users to input numeric values: either integer or floating point numbers. It also accepts empty values, where the stored value will be of type `null`.

The `NumberBox` is an input control component and is often used in forms. See the [Using Forms](/learning/using-components/forms/) guide for details.

## Context Variables

This component does not have any context variables.

## Styling

The `NumberBox` component uses these theme variables to customize its appearance:

- `radius-NumberBox`
- `color-text-NumberBox`
- `color-bg-NumberBox--disabled`
- `thickness-border-NumberBox`
- `style-border-NumberBox`
- `color-border-NumberBox--disabled`
- `color-bg-NumberBox--disabled`
- `color-text-NumberBox--disabled`
- `color-border-NumberBox-default`
- `color-border-NumberBox-error`
- `color-border-NumberBox-warning`
- `color-border-NumberBox-success`
- `color-placeholder-NumberBox`

```json
"radius-NumberBox": "30px",
"color-adornment-NumberBox": "red",
"thickness-border-NumberBox": "4px"
```

<Playground

To style all input controls not just this component, use `Input` instead of the `NumberBox` segment:

```
color-border-Input: "#0033FF"
```

`Input` affects the following controls:

- [`Checkbox`](./Checkbox.mdx)
- [`RadioGroupOption`](./RadioGroupOption.mdx)
- [`Select`](./Select.mdx)
- [`Switch`](./Switch.mdx)
- [`TextArea`](./TextArea.mdx)
- [`TextBox`](./TextBox.mdx)

See [custom themes](../learning/themes/overview.mdx) for styling details.
