# PageHeader

The `PageHeader` component is a component that displays a title and an optional pre-title. The pre-title is displayed above the title.

## Context Variables

This component does not have any context variables.

## Properties

### `preTitle`

The pre-title to display above the title.

### `title`

The title to display.
