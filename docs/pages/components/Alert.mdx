# Alert

(**NOT IMPLEMENTED YET**) The `Alert` component is a panel used to display important notifications or feedback to users. It helps convey different statuses or levels of urgency, such as success, warning, error, and others.

## Context Variables

This component does not have any context variables.

## Properties

### `autoDismissInMs`

Timeout for the alert to be dismissed

### `dismissable`

This property's `true` value indicates if this alert is dismissable by the user. When the user closes the Alert, it gets hidden.

### `icon`

Icon to be displayed in the alert

### `showIcon`

Indicates if the alert should display an icon

### `statusColor`

The value of this optional property sets the string to provide a color scheme for the Alert.

Default value: `"primary"`.

Available values:

| Value | Description |
| --- | --- |
| `primary` | Primary theme color, no default icon **(default)** |
| `secondary` | Secondary theme color, no default icon |
| `success` | Success theme color, "success" icon |
| `danger` | Warning theme color, "warning" icon |
| `warning` | Danger theme color, "danger" icon |
| `info` | Info theme color, "info" icon |
| `light` | Light theme color, no default icon |
| `dark` | Dark theme color, no default icon |


The value of this optional property sets the string to provide a color scheme for the Alert.
