# HStack [#component-hstack]

This component is inherited from <SmartLink href="/components/Stack">Stack</SmartLink>

See also: <SmartLink href="/components/CHStack">CHStack</SmartLink>, <SmartLink href="/components/CVStack">CVStack</SmartLink>, <SmartLink href="/components/VStack">VStack</SmartLink>

`HStack` is a specialized, shorthand version for the regular `Stack` component with a horizontal orientation.

```ueml copy
<HStack gap="1rem">
  <Stack height="32px" width="32px" backgroundColor="red" />
  <Stack height="32px" width="32px" backgroundColor="blue" />
  <Stack height="32px" width="32px" backgroundColor="green" />
</HStack>
```

 `}

> **Note**: You cannot change the orientation of `HStack` from horizontal to vertical by setting the `orientation` prop, as the engine ignores that setting.
