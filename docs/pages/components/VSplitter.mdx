# VSplitter [#component-vsplitter]

This component is inherited from <SmartLink href="/components/Splitter">Splitter</SmartLink>

See also: <SmartLink href="/components/HSplitter">HSplitter</SmartLink>

`VSplitter` is a specialized, shorthand version for the regular `Splitter` component with a vertical orientation.

```ueml copy
<App height="100%">
  <VSplitter>
    <Stack backgroundColor="lightblue" height="100%" />
    <Stack backgroundColor="darksalmon" height="100%" />
  </VSplitter>
</App>
```

> **Note**: You cannot change the orientation of `VSplitter` from vertical to horizontal by setting the `orientation` prop, as the engine ignores that setting.
