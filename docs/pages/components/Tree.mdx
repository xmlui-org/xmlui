# Tree

The `Tree` component is a virtualized tree component that displays hierarchical data.

## Properties

### `data`

The data source of the tree.

### `itemTemplate`

The template for each item in the tree.

### `selectedUid`

The ID (optional) of the selected tree row.

## Events

This component does not have any events.
