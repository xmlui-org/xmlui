This component renders an HTML `th` tag.



## Properties

### `abbr`

Specifies an abbreviated version of the content in the header cell

### `align`

Specifies the horizontal alignment of the content in the header cell

### `colSpan`

Specifies the number of columns a header cell should span

### `headers`

Specifies a list of header cells the current header cell is related to

### `rowSpan`

Specifies the number of rows a header cell should span

### `scope`

Specifies whether a header cell is a header for a column, row, or group of columns or rows
