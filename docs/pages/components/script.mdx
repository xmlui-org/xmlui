This component renders an HTML `script` tag.



## Properties

### `async`

Specifies that the script should be executed asynchronously

### `crossOrigin`

Specifies how the script should handle cross-origin requests

### `defer`

Specifies that the script should be executed after the document has been parsed

### `integrity`

Specifies the integrity of the script

### `noModule`

Specifies that the script should not be executed in module-supporting browsers

### `referrerPolicy`

Specifies the referrer policy for the script

### `src`

Specifies the URL of the external script file

### `type`

Specifies the MIME type of the script
