This component renders an HTML `td` tag.



## Properties

### `abbr`

Specifies an abbreviated version of the content in the cell

### `align`

Specifies the horizontal alignment of the content in the cell

### `colSpan`

Specifies the number of columns a cell should span

### `headers`

Specifies a list of header cells the current cell is related to

### `height`

Specifies the height of the cell

### `rowSpan`

Specifies the number of rows a cell should span

### `scope`

Specifies whether a cell is a header for a column, row, or group of columns or rows

### `valign`

Specifies the vertical alignment of the content in the cell

### `width`

Specifies the width of the cell
