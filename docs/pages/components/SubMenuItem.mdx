# SubMenuItem

This component represents a submenu within a menu hierarchy. When the user moves the mouse over the submenu, its content is displayed. See the [`DropdownMenu` component](./DropdownMenu) for using this component in menus.

## Properties

### `label`

This property defines the submenu's label.

### `triggerTemplate`

This property allows you to define a custom submenu trigger instead of the default one provided by `SubMenuItem`:

```ueml copy {9}
<DropdownMenu label="DropdownMenu">
  <MenuItem>Item 1</MenuItem>
  <MenuItem>Item 2</MenuItem>
  <MenuSeparator />
  <SubMenuItem label="Submenu">
    <prop name="triggerTemplate">
      <HStack padding="1rem" backgroundColor="cyan">
        Submenu
      </HStack>
    </prop>  
    <MenuItem>Submenu Item 1</MenuItem>
    <MenuItem>Submenu Item 2</MenuItem>
  </SubMenuItem>
</DropdownMenu>
```

## Events

This component does not have any events.

## Styling

The `SubMenuItem` component uses these theme variables to customize its appearance:

- `color-bg`
- `min-width`
- `radius`
- `shadow`
