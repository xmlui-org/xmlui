# input [#component-input]

<Callout type="warning" emoji="ðŸ“”">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

This component renders an HTML `input` tag.



## Properties

### `autoFocus`

Specifies that the input should automatically get focus when the page loads

### `checked`

Specifies that the input should be pre-selected

### `disabled`

Specifies that the input should be disabled

### `form`

Specifies the form the input belongs to

### `list`

Specifies the id of a datalist element that contains pre-defined options for the input

### `max`

Specifies the maximum value for an input

### `maxLength`

Specifies the maximum number of characters allowed in an input

### `min`

Specifies the minimum value for an input

### `minLength`

Specifies the minimum number of characters allowed in an input

### `multiple`

Specifies that a user can enter more than one value

### `name`

Specifies the name of the input

### `pattern`

Specifies a regular expression that an input's value is checked against

### `placeholder`

Specifies a short hint that describes the expected value of the input

### `readOnly`

Specifies that the input is read-only

### `required`

Specifies that the input is required

### `size`

Specifies the width, in characters, of an input

### `step`

Specifies the legal number intervals for an input

### `type`

Specifies the type of input

### `value`

Specifies the value of the input
