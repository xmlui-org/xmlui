# Pdf

The `Pdf` component provides a read-only preview of a pdf document's contents.

## Context Variables

This component does not have any context variables.

## Properties

### `src`

This property defines the source URL of the pdf document stream to display.

```ueml copy
<VStack>
  <Pdf src="https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" />
</VStack>
```
