# Tabs

The `Tabs` component provides a tabbed layout where each tab has a clickable label and content.
The label is displayed on the button associated with the content.

The component accepts only `TabItem` components as children.
Other child components will not be displayed.

The `TabItem` component has a `label` prop to define the tab button label text.
Content is provided to a tab via placing child components under the `TabItem`.

```ueml copy
<Tabs>
  <TabItem label="Account">
    <Text>Account</Text>
  </TabItem>
  <TabItem label="Stream">
    <Text>Stream</Text>
  </TabItem>
  <TabItem label="Support">
    <Text>Support</Text>
  </TabItem>
</Tabs>
```

This component does not have any context variables.

## Properties

### `activeTab`

Indicates which tab index is active. The indexing starts from 0. The default value is 0 (aka the first tab from left to right).

```ueml copy /activeTab="2"/
<Tabs activeTab="2">
  <TabItem label="Account">
    <Text>Account</Text>
  </TabItem>
  <TabItem label="Stream">
    <Text>Stream</Text>
  </TabItem>
  <TabItem label="Support">
    <Text>Support</Text>
  </TabItem>
</Tabs>
```

### `orientation`

Indicates the orientation of the tabs.
Horizontal orientation has the tab buttons layed out on the left side of the content panel.
Vertical orientation places the buttons at the top.
The default value is vertical.

```ueml copy /orientation="horizontal"/
<Tabs orientation="horizontal">
  <TabItem label="Account">
    <Text>Account</Text>
  </TabItem>
  <TabItem label="Stream">
    <Text>Stream</Text>
  </TabItem>
  <TabItem label="Support">
    <Text>Support</Text>
  </TabItem>
</Tabs>
```

## Styling

The `Tabs` component uses these theme variables to customize its appearance:

- `color-bg-Tabs`
- `color-border-Tabs`
- `thickness-border-Tabs`
- `color-border-active-Tabs`
- `color-bg-trigger-Tabs`
- `color-bg-trigger-Tabs--hover`

```json copy
"color-bg-Tabs": "green",
"thickness-border-Tabs": "6px",
"color-border-Tabs": "gold"
```
