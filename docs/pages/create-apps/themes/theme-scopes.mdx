# Theme Scopes

There are situations when your app wants to apply multiple themes for the same page: while most of the page is displayed with the selected theme, particular sections use a different one.

Often, apps display their headers in a contrasting tone; when the app's body is in a light tone, the header is displayed in a dark:

<Playground 
  previewOnly 
  name="Multi-theme app"
  height={350}
  defaultTone="light" 
  app={`
    <App layout="condensed-sticky">
      <Theme tone="dark">
        <AppHeader>
            <Heading level="h3" value="Example App"/>
        </AppHeader>
        <NavPanel>
          <NavLink label="Home" to="/" icon="home"/>
          <NavLink label="Page 1" to="/page1"/>
          <NavLink label="Page 2" to="/page2"/>
        </NavPanel>
      </Theme>  
      <Pages defaultRoute="/">
        <Page url="/">
          <List datasource="https://api.spacexdata.com/v3/history">
            <property name="itemTemplate">
              <Card title="{$item.title}" subTitle="{$item.details}"/>
            </property>
          </List>
        </Page>
        <Page url="/page1">
          <Text value="Page 1" />
        </Page>
        <Page url="/page2">
          <Text value="Page 2" />
        </Page>
      </Pages>
      <Footer>Powered by XMLUI</Footer>
    </App>
  `} 
/>

XMLUI has a powerful component, `Theme,` that you can use to provide a particular theming context for a markup segment. The example above uses `Theme` to wrap the application header and the navigation bar into a dark tone context:

```ueml copy {2-11}
<App layout="condensed-sticky">
  <Theme tone="dark">
    <AppHeader>
        <Heading level="h3" value="Example App"/>
    </AppHeader>
    <NavPanel>
      <NavLink label="Home" to="/" icon="home"/>
      <NavLink label="Page 1" to="/page1"/>
      <NavLink label="Page 2" to="/page2"/>
    </NavPanel>
  </Theme>  
  <Pages defaultRoute="/">
    <Page url="/">
      <List datasource="https://api.spacexdata.com/v3/history">
        <property name="itemTemplate">
          <Card title="{$item.title}" subTitle="{$item.details}"/>
        </property>
      </List>
    </Page>
    <Page url="/page1">
      <Text value="Page 1" />
    </Page>
    <Page url="/page2">
      <Text value="Page 2" />
    </Page>
  </Pages>
  <Footer>Powered by XMLUI</Footer>
</App>
```

You can wrap any markup with the `Theme` component to change its theming context. For using this component, see the [`Theme` reference](../../components/Theme).
