# Basic Components

<Callout type="warning" emoji="⚠️">
  This article is under construction. We are continuously adding components to it.

This article contains an introduction to basic components in XMLUI.
Component features are presented with examples.

## Button

Use the `Button` component to let users trigger various actions on the UI.

```ueml copy
<App>
  <Button label="Click me!" onClick="toast('Button clicked')" />
  <Button label="Can't Click Here" enabled="{false}" />
</App>
```

## Checkbox

The `Checkbox` component allows users to make binary choices by toggling an input control on or off.

It works the same way as the [Switch](#switch) component - the only difference is that the `Checkbox` also has an indeterminate state.

```ueml copy 
<App>
  <HStack gap="$space-2">
    Enabled checkboxes:
    <Checkbox initialValue="true" enabled="true" />
    <Checkbox initialValue="false" enabled="true" />
  </HStack>
  <HStack gap="$space-2">
    Disabled checkboxes:
    <Checkbox initialValue="true" enabled="false" />
    <Checkbox initilaValue="false" enabled="false" />
  </HStack>
</App>
```

<Playground
  previewOnly
  name="Example: Checkboxes"
  app={`
    <App>
      <HStack gap="$space-2">
        Enabled checkboxes:
        <Checkbox initialValue="true" enabled="true" />
        <Checkbox initialValue="false" enabled="true" />
      </HStack>
      <HStack gap="$space-2">
        Disabled checkboxes:
        <Checkbox initialValue="true" enabled="false" />
        <Checkbox initilaValue="false" enabled="false" />
      </HStack>
    </App>
  `}
/>

The `Checkbox` component can indicate the state of a group of other checkboxes.
This state is visual only, the actual value of the checkbox is still either `true` or `false`.

The demo app below uses an XNOR operator as an example.

<Callout type="info" emoji="💡">
  Get the data of an input control by assigning an ID to it and using dot notation to access its `value`.
</Callout>

```ueml copy {3}
<App>
  <Checkbox
    indeterminate="{![item1.value, item2.value, item3.value].every((value) => value === item1.value)}"
    label="Indeterminate Indicator"
    initialValue="true"
    readOnly="true" />
  <Card>
    <HStack>
      <Checkbox id="item1" initialValue="true" />
      <Checkbox id="item2" initialValue="false" />
      <Checkbox id="item3" initialValue="true" />
    </HStack>
  </Card>
</App>
```

## RadioGroup

This component can contain a set of radio group buttons (`RadioGroupOption`) and enable the selection of only one of them at any one time.

```ueml copy
<App>
  <RadioGroup initialValue="first">
    <HStack gap="$space-2">
      <RadioGroupOption label="First Item" value="first"/>
      <RadioGroupOption label="Second Item" value="second"/>
      <RadioGroupOption label="Third Item" value="third"/>
    </HStack>
  </RadioGroup>
</App>
```

<Playground
  previewOnly
  name="Example: RadioGroup"

          <RadioGroupOption label="Second Item" value="second"/>
          <RadioGroupOption label="Third Item" value="third"/>

      </RadioGroup>
    </App>
  `}
/>

        <RadioGroupOption label="Third Item" value="third" />

    </RadioGroup>
    <RadioGroup initialValue="first" enabled="false">
      <VStack gap="$space-2">
        <RadioGroupOption label="First Item" value="first"/>
        <RadioGroupOption label="Second Item" value="second"/>
        <RadioGroupOption label="Third Item" value="third"/>
      </VStack>
    </RadioGroup>
  </HStack>
</App>

      <RadioGroupOption label="First Item" value="first"/>

        <RadioGroupOption label="Second Item" value="second"/>

      <RadioGroupOption label="Third Item" value="third"/>
      <RadioGroup initialValue="second">
        Group #2
        <Card gap="$space-2">
          <RadioGroupOption label="First Item" value="first"/>
          <RadioGroupOption label="Second Item" value="second"/>

## Combobox

This component provides a list of selectable options and the ability to filter these options using its input field so that one can select an item from the list.
For selecting multiple items from a list, refer to the [Multicombobox component](#multicombobox).

```ueml copy 
<App>
  <HStack gap="$space-loose">
    <Combobox>
      <Option label="Jane Arrington" value="ja" />
      <Option label="John Sniper" value="js" />
      <Option label="Emily Gold" value="eg" />
    </Combobox>
    <Combobox>
      <Items items="{['Jane', 'John', 'Emily']}">
        <Option label="Option {$item}" value="{$item}" />
      </Items>
    </Combobox>
  </HStack>
</App>
```

## DatePicker

Enables the selection of a date or a range of dates in a specified format from an interactive display.

```ueml copy
<App>
  <DatePicker dateFormat="dd/MM/yyyy" initialValue="25/05/2024" />
  <DatePicker dateFormat="yyyy-MM-dd" initialValue="25/05/2024" />
  <DatePicker dateFormat="MMddyyyy" initialValue="25/05/2024" />
</App>
```

<Playground
  previewOnly
  name="Example: DatePicker and Date Formatting"
  height={260}
  app={`
    <App>
      <DatePicker dateFormat="dd/MM/yyyy" initialValue="25/05/2024" />
      <DatePicker dateFormat="yyyy-MM-dd" initialValue="25/05/2024" />
      <DatePicker dateFormat="MMddyyyy" initialValue="25/05/2024" />
    </App>
  `}
/>

<Callout type="info" emoji="💡">
  Use the `mode` property to specify the number of selectable dates: "single" or "range".
</Callout>

```ueml copy
<App>
  <DatePicker initialValue="25/05/2024" mode="single" />
  <DatePicker initialValue="{{ from: '25/05/2024', to: '25/07/2024' }}" mode="range" />
</App>
```

## Multicombobox

This component provides a list of selectable options and the ability to filter these options using its input field so that one can select a subset of the original list.

For selecting only one item from a list, refer to the [Combobox component](#combobox).

```ueml copy
<App>
  <HStack gap="$space-loose">
    <Multicombobox>
      <Option label="Jane Arrington" value="ja" />
      <Option label="John Sniper" value="js" />
      <Option label="Emily Gold" value="eg" />
    </Multicombobox>
    <Multicombobox>
      <Items items="{['Jane', 'John', 'Emily']}">
        <Option label="Option {$item}" value="{$item}" />
      </Items>
    </Multicombobox>
  </HStack>
</App>
```
