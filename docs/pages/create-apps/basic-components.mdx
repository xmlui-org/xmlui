# Basic Components

## Button

See <SmartLink href={COMPONENT_BUTTON}>component reference</SmartLink>

Use the `Button` component to let users trigger various actions on the UI.

```ueml copy
<HStack>
  <Button label="Click me!" onClick="window.alert('Button clicked')" />
  <Button label="Can't Click Here" enabled="{false}" />
</HStack>
```

<Callout type="info" emoji="ðŸ’¡">
  Properties such as `variant`, `themeColor` and `size` provide ways to emphasize certain actions.
</Callout>

```ueml copy {7, 10-11}
<Card>
  <TextBox id="field" initialValue="John Doe" />
  <CHStack>
    <Button
      label="Clear Text Field"
      onClick="field.setValue('')"
      size="xs" variant="ghost" themeColor="attention"
    />
    <SpaceFiller />
    <Button label="Cancel" variant="outlined" themeColor="secondary" />
    <Button label="Update" onClick="toast('Updated value: ' + field.value)" />
  </CHStack>
</Card>
```

<Callout type="info" emoji="ðŸ’¡">
  Buttons can also have icons to provide visual cues for actions.
</Callout>

```ueml copy
<HStack>
  <Button icon="close" variant="ghost" themeColor="secondary" />
  <Button label="Clear Text Field" icon="trash" iconPosition="end" variant="outlined" themeColor="secondary" />
  <Button label="Update" icon="pen" />
</HStack>
```
