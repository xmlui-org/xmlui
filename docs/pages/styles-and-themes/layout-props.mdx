# Layout Properties

This document summarizes the layout properties you can use for UI Engine components.

## `backgroundColor`

Sets the background [color](./common-units/#color-values) of a component.

## `border`

Sets the value of the [border](./common-units/#border-values) area; uses the same value for the left, top, right, and bottom borders.

## `borderBottom`

Sets the value of the bottom [border](./common-units/#border-values) to the specified one.

## `borderColor`

Sets the [color](./common-units/#color-values) of the border; uses the same value for the left, top, right, and bottom borders.

## `borderLeft`

Sets the value of the left [border](./common-units/#border-values) to the specified one.

## `borderRight`

Sets the value of the right [border](./common-units/#border-values) to the specified one.

## `borderStyle`

Sets the [style](./common-units/#border-style-values) of the border; uses the same value for the left, top, right, and bottom borders.

## `borderThickness`

Sets the [thickness](./common-units/#size-values) of the border; uses the same value for the left, top, right, and bottom borders.

## `borderTop`

Sets the value of the top [border](./common-units/#border-values) to the specified one.

## `bottom`

Sets the [bottom](./common-units/#size-values) position of the component to the specified value.

## `color`

Sets the foreground [color](./common-units/#color-values) of a component's text and text decorations.

## `cursor`

Sets the mouse [cursor](./common-units/#cursor-values), if any, to show when the mouse pointer is over a particular component

## `direction`

This property sets the direction of text, table columns, and horizontal overflow. Use `rtl` for languages written from right to left (like Hebrew or Arabic), and `ltr` for those written from left to right (like English and most other languages). By default, the prop's value is `ltr`.

## `fontFamily`

This property specifies a prioritized list of one or more [font family](./common-units/#font-family-values) names and/or generic family names for the selected component.

## `fontSize`

This property sets the size of the font. Changing the font [size](./common-units/#size-values) also updates the sizes of the font size-relative length units, such as em, ex, and so forth.

## `fontWeight`

Sets the [weight (or boldness)](./common-units/#font-weight-values) of the font. The weights available depend on the `fontFamily` that is currently set.

## `gap`

Layout components use this property to define the [size](./common-units/#size-values) of space separating child components.

## `height`

Specifies the height (vertical dimension) of the component.

## `horizontalAlignment`

This property specifies the horizontal [alignment](./common-units/#alignment-values) of child component within its parent.

## `horizontalMargin`

Sets the [size](./common-units/#size-values) of the horizontal margins (left and right) to the specified value.

## `horizontalPadding`

Sets the [size](./common-units/#size-values) of the horizontal padding (left and right) to the specified value.

## `horizontalOverflow`

This property defines the [overflow](./common-units/#overflow-values) strategy when a component horizontally overflows the UI its parent provides.

## `italic`

This [Boolean](./common-units/#boolean-values) property determines if the text should be displayed in italic style.

## `left`

Sets the [left](./common-units/#size-values) position of the component to the specified value.

## `letterSpacing`

This property sets the horizontal [spacing](./common-units/#size-values) behavior between text characters. This value is added to the natural spacing between characters while rendering the text. Positive values cause characters to spread farther apart, while negative values bring characters closer together.

## `lineHeight`

This property sets the [height](../reference/styling/common-units/#size-values) of a line box. It's commonly used to set the distance between lines of text.

## `margin`

Sets the [size](./common-units/#size-values) of the margin area; uses the same value for the left, top, right, and bottom margins.

## `marginBottom`

Sets the [size](./common-units/#size-values) of the bottom margin to the specified value.

## `marginLeft`

Sets the [size](./common-units/#size-values) of the left margin to the specified value.

## `marginRight`

Sets the [size](./common-units/#size-values) of the right margin to the specified value.

## `marginTop`

Sets the [size](./common-units/#size-values) of the top margin to the specified value.

## `maxHeight`

Specifies the maximum height of an element. It prevents the current value of the height property from becoming larger than the value specified.

## `maxWidth`

Specifies the maximum width of an element. It prevents the current value of the width property from becoming larger than the value specified.

## `minHeight`

Specifies the minimum height of an element. It prevents the current value of the height property from becoming smaller than the value specified.

## `minWidth`

Specifies the minimum width of an element. It prevents the current value of the width property from becoming smaller than the value specified.

## `opacity`

This property sets the opacity of an element. Opacity is the degree to which the content behind an element is hidden. The value is a number in the range 0.0 to 1.0, inclusive, or a percentage from 0% to 100%.

## `orientation`

This property specifies a particular component's [orientation](./common-units/#orientation) when displaying its child components.

## `padding`

Sets the [size](./common-units/#size-values) of the padding area; uses the same value for the left, top, right, and bottom paddings.

## `paddingBottom`

Sets the [size](./common-units/#size-values) of the bottom padding to the specified value.

## `paddingLeft`

Sets the [size](./common-units/#size-values) of the left padding to the specified value.

## `paddingRight`

Sets the [size](./common-units/#size-values) of the right padding to the specified value.

## `paddingTop`

Sets the [size](./common-units/#size-values) of the top padding to the specified value.

## `radius`

This property rounds the corners of a component's outer border edge according to the specified [value](./common-units/#border-rounding).

## `radiusBottomLeft`

This property rounds the bottom-left corner of a component's outer border edge according to the specified [value](./common-units/#border-rounding).

## `radiusBottomRight`

This property rounds the bottom-right corner of a component's outer border edge according to the specified [value](./common-units/#border-rounding).

## `radiusTopLeft`

This property rounds the top-left corner of a component's outer border edge according to the specified [value](./common-units/#border-rounding).

## `radiusTopRight`

This property rounds the top-right corner of a component's outer border edge according to the specified [value](./common-units/#border-rounding).

## `right`

Sets the [right](./common-units/#size-values) position of the component to the specified value.

## `shadow`

This property adds [shadow effects](./common-units/#shadow-values) around a component's frame and set up one or more shadow effects.

## `textAlign`

This property sets the horizontal [alignment](./common-units/#text-alignment-values) of the inline-rendered content (e.g., text) inside its block-rendered parent.

## `textAlignLast`

This property sets how the last line of a block or a line, right before a forced line break, is [aligned](./common-units/#text-alignment-values).

## `textDecoration`

This property sets the appearance of [decorative lines](./common-units/#text-decoration-values) on text.

## `textTransform`

This property specifies how to [capitalize](./common-units/#text-transform-values) an element's text.

## `top`

Sets the [top](./common-units/#size-values) position of the component to the specified value.

## `userSelect`

This property controls whether the [user can select](./common-units/#user-select-values) text.

## `verticalAlignment`

This property specifies the vertical [alignment](./common-units/#alignment-values) of child component within its parent.

## `verticalMargin`

Sets the [size](./common-units/#size-values) of the vertical margins (top and bottom) to the specified value.

## `verticalPadding`

Sets the [size](./common-units/#size-values) of the vertical paddings (top and bottom) to the specified value.

## `verticalOverflow`

This property defines the [overflow](./common-units/#overflow-values) strategy when a component vertically overflows the UI its parent provides.

## `width`

Specifies the width (horizontal dimension) of the component.

## `zIndex`

This layout property sets the z-order of a positioned component and its children. Overlapping components with a larger z-index cover those with a smaller one.
