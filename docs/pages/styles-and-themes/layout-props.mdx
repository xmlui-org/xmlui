# Layout Properties [#layout-summary]

This document summarizes the layout properties you can use for UI Engine components.

## `backgroundColor`

Sets the background <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#color-values"}>color value</SmartLink> of a component.

## `border`

Sets the value of the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-values"}>border</SmartLink> area; uses the same value for the left, top, right, and bottom borders.

## `borderBottom`

Sets the value of the bottom <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-values"}>border</SmartLink> to the specified one.

## `borderColor`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#color-values"}>color value</SmartLink> of the border; uses the same value for the left, top, right, and bottom borders.

## `borderLeft`

Sets the value of the left <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-values"}>border</SmartLink> to the specified one.

## `borderRight`

Sets the value of the right <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-values"}>border</SmartLink> to the specified one.

## `borderStyle`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-style-values" }>style</SmartLink> of the border; uses the same value for the left, top, right, and bottom borders.

## `borderThickness`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>thickness</SmartLink> of the border; uses the same value for the left, top, right, and bottom borders.

## `borderTop`

Sets the value of the top <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-values"}>border</SmartLink> to the specified one.

## `bottom`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values"}>bottom</SmartLink> position of the component to the specified value.

## `color`

Sets the foreground <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#color-values"}>color value</SmartLink> of a component's text and text decorations.

## `cursor`

Sets the mouse <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#cursor-values" }>cursor</SmartLink>, if any, to show when the mouse pointer is over a particular component

## `direction`

This property sets the direction of text, table columns, and horizontal overflow. Use `rtl` for languages written from right to left (like Hebrew or Arabic), and `ltr` for those written from left to right (like English and most other languages). By default, the prop's value is `ltr`.

## `fontFamily`

This property specifies a prioritized list of one or more <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#font-family-values" }>font family</SmartLink> names and/or generic family names for the selected component.

## `fontSize`

This property sets the size of the font. Changing the font <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> also updates the sizes of the font size-relative length units, such as em, ex, and so forth.

## `fontWeight`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#font-weight-values" }>weight (or boldness)</SmartLink> of the font. The weights available depend on the `fontFamily` that is currently set.

## `gap`

Layout components use this property to define the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of space separating child components.

## `height`

Specifies the height (vertical dimension) of the component.

## `horizontalAlignment`

This property specifies the horizontal <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#alignment-values" }>alignment</SmartLink> of child component within its parent.

## `horizontalMargin`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values"}>size</SmartLink> of the horizontal margins (left and right) to the specified value.

## `horizontalPadding`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values"}>size</SmartLink> of the horizontal padding (left and right) to the specified value.

## `horizontalOverflow`

This property defines the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#overflow-values" }>overflow</SmartLink> strategy when a component horizontally overflows the UI its parent provides.

## `italic`

This <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#boolean-values" }>boolean</SmartLink> property determines if the text should be displayed in italic style.

## `left`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>left</SmartLink> position of the component to the specified value.

## `letterSpacing`

This property sets the horizontal <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>spacing</SmartLink> behavior between text characters. This value is added to the natural spacing between characters while rendering the text. Positive values cause characters to spread farther apart, while negative values bring characters closer together.

## `lineHeight`

This property sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>height</SmartLink> of a line box. It's commonly used to set the distance between lines of text.

## `margin`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the margin area; uses the same value for the left, top, right, and bottom margins.

## `marginBottom`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the bottom margin to the specified value.

## `marginLeft`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the left margin to the specified value.

## `marginRight`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the right margin to the specified value.

## `marginTop`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the top margin to the specified value.

## `maxHeight`

Specifies the maximum height of an element. It prevents the current value of the height property from becoming larger than the value specified.

## `maxWidth`

Specifies the maximum width of an element. It prevents the current value of the width property from becoming larger than the value specified.

## `minHeight`

Specifies the minimum height of an element. It prevents the current value of the height property from becoming smaller than the value specified.

## `minWidth`

Specifies the minimum width of an element. It prevents the current value of the width property from becoming smaller than the value specified.

## `opacity`

This property sets the opacity of an element. Opacity is the degree to which the content behind an element is hidden. The value is a number in the range 0.0 to 1.0, inclusive, or a percentage from 0% to 100%.

## `orientation`

This property specifies a particular component's <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#orientation" }>orientation</SmartLink> when displaying its child components.

## `padding`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the padding area; uses the same value for the left, top, right, and bottom paddings.

## `paddingBottom`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the bottom padding to the specified value.

## `paddingLeft`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the left padding to the specified value.

## `paddingRight`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the right padding to the specified value.

## `paddingTop`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the top padding to the specified value.

## `radius`

This property rounds the corners of a component's outer border edge according to the specified <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-rounding" }>value</SmartLink>.

## `radiusBottomLeft`

This property rounds the bottom-left corner of a component's outer border edge according to the specified <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-rounding" }>value</SmartLink>.

## `radiusBottomRight`

This property rounds the bottom-right corner of a component's outer border edge according to the specified <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-rounding" }>value</SmartLink>.

## `radiusTopLeft`

This property rounds the top-left corner of a component's outer border edge according to the specified <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-rounding" }>value</SmartLink>.

## `radiusTopRight`

This property rounds the top-right corner of a component's outer border edge according to the specified <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#border-rounding" }>value</SmartLink>.

## `right`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>right</SmartLink> position of the component to the specified value.

## `shadow`

This property adds <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#shadow-values" }>shadow effects</SmartLink> around a component's frame and set up one or more shadow effects.

## `textAlign`

This property sets the horizontal <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#text-alignment-values" }>alignment</SmartLink> of the inline-rendered content (e.g., text) inside its block-rendered parent.

## `textAlignLast`

This property sets how the last line of a block or a line, right before a forced line break, is <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#text-alignment-values" }>aligned</SmartLink>.

## `textDecoration`

This property sets the appearance of <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#text-decoration-values" }>decorative lines</SmartLink> on text.

## `textTransform`

This property specifies how to <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#text-transform-values" }>capitalize</SmartLink> an element's text.

## `top`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>top</SmartLink> position of the component to the specified value.

## `userSelect`

This property controls whether the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#user-select-values" }>user can select</SmartLink> text.

## `verticalAlignment`

This property specifies the vertical <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#alignment-values" }>alignment</SmartLink> of child component within its parent.

## `verticalMargin`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the vertical margins (top and bottom) to the specified value.

## `verticalPadding`

Sets the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#size-values" }>size</SmartLink> of the vertical paddings (top and bottom) to the specified value.

## `verticalOverflow`

This property defines the <SmartLink href={ COMMON_VISUAL_PROPERTY_UNITS + "#overflow-values" }>overflow</SmartLink> strategy when a component vertically overflows the UI its parent provides.

## `width`

Specifies the width (horizontal dimension) of the component.

## `zIndex`

This layout property sets the z-order of a positioned component and its children. Overlapping components with a larger z-index cover those with a smaller one.
