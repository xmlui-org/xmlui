# ScaleAnimation [#component-scaleanimation]

<Callout type="warning" emoji="ðŸ“”">This component is in an **experimental** state; you can use it in your app. However, we may modify it, and it may even have breaking changes in the future.</Callout>

The `ScaleAnimation` component represents an animation that scales the content.

## Properties

### `animateWhenInView`

Indicates whether the animation should start when the component is in view.

```ueml copy /animateWhenInView/
<App>
  <Button onClick="anim.start()">Start</Button>
  <ScaleAnimation id="anim" from="0" to="1" animateWhenInView>
    <Image
      src="/resources/images/components/image/breakfast.jpg" width="500"/>
  </ScaleAnimation>
</App>
```
