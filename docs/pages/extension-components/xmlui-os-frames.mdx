# XMLUI OS Frames Package

This package provides containers that appear to look like OS-specific windows.

## Package Components

| Component | Description | Status |
| :---: | --- | :---: |
| [IphoneFrame](./xmlui-os-frames/IphoneFrame.mdx) | App Frame for IPhone. | experimental |
| [MacAppFrame](./xmlui-os-frames/MacAppFrame.mdx) | App Frame for MacOS. | experimental |
| [WindowsAppFrame](./xmlui-os-frames/WindowsAppFrame.mdx) | App Frame for Windows. | experimental |
