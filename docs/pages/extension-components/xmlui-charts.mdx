## Xmlui Charts Package

The `Charts` package contains components that display data in a graphical way.All chart components use the same set of properties to receive data.All chart components accept a `LabelLst` component as a child to parametrize display labels.All chart components also accept a `Legend` component as a child to display the legend.

## Package Components

| Component | Description | Status |
| :---: | --- | :---: |
| [BarChart](./xmlui-charts/BarChart.mdx) | The `BarChart` component represents a bar chart.Accepts a `LabelLst` component as a child to parametrize display labels. | experimental |
| [DonutChart](./xmlui-charts/DonutChart.mdx) | Represents a derivative of the pie chart that is a donut chart. | stable |
| [LabelList](./xmlui-charts/LabelList.mdx) | Label list component for a chart component. | stable |
| [Legend](./xmlui-charts/Legend.mdx) | Legend component to be displayed in a chart component. | stable |
| [LineChart](./xmlui-charts/LineChart.mdx) | Represents a line chart component. | stable |
| [PieChart](./xmlui-charts/PieChart.mdx) | Represents a pie chart component. | stable |
