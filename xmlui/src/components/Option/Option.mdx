## Using `Option`

### With `Combobox`

```ueml copy {4-6}
<VStack padding="$space-4" gap="$space-4">
  <Text value="Selected ID: {myCombo.value}"/>
  <Combobox id="myCombo">
    <Option label="John, Smith" value="john" />
    <Option label="Jane, Clint" value="jane" />
    <Option label="Herbert, Frank" value="herbert" />
  </Combobox>
</VStack>
```

%-PROP-START label

> **Note**: If `Option` does not define any of the `label` or `value` properties, the option will not be rendered.

```ueml copy
<VStack padding="$space-4" gap="$space-4">
  <Text value="Selected ID: {mySelect.value}"/>
  <Select id="mySelect">
    <Option />
    <Option label="Vanilla" value="van"/>
    <Option label="Chocolate" value="choc" />
    <Option value="pist" />
  </Select>
</VStack>
```
