# NavPanel

`NavPanel` is a placeholder within `App` to define the app's navigation (menu) structure.

> **Note**: You can learn more details about using this component [here](../learning/app-component).

%-PROP-START logoTemplate

```ueml copy {3-8}
<App layout="vertical">
  <NavPanel>
    <property name="logoTemplate">
       <H3>
         <Icon name="drive" />
         DriveDiag (Nav)
       </H3>
    </property>
    <NavLink label="Home" to="/" icon="home"/>
    <NavLink label="Page 1" to="/page1"/>
  </NavPanel>
  <Pages defaultRoute="/">
    <Page url="/">
      <Text value="Home" />
    </Page>
    <Page url="/page1">
      <Text value="Page 1" />
    </Page>
  </Pages>
</App>
```

%-STYLE-START

The `NavPanel` component uses these theme variables to customize its appearance:

- `color-bg`
- `boxShadow`
- `paddingHorizontal`
- `paddingHorizontal-logo-NavPanel`
- `paddingVertical-logo-NavPanel`
- `margin-bottom-logo-NavPanel`
