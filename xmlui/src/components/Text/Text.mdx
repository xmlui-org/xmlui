**HtmlTag Mappings**

The table below indicates which Text `variant` maps to which HtmlTag component.

| Variant     | Component |
| ----------- | --------- |
| `abbr`      | abbr      |
| `cite`      | cite      |
| `code`      | code      |
| `codefence` | pre       |
| `deleted`   | del       |
| `inserted`  | ins       |
| `keyboard`  | kbd       |
| `marked`    | mark      |
| `sample`    | samp      |
| `sub`       | sub       |
| `sup`       | sup       |
| `var`       | var       |
| `strong`    | strong    |
| `em`        | em        |
| `paragraph` | p         |

%-STYLE-START

The `Text` component uses these theme variables:

- `align-vertical`
- `borderRadius`
- `color`
- `color-bg`
- `color-border`
- `textDecorationColor`
- `fontFamily`
- `fontSize`
- `fontWeight`
- `font-style`
- `font-stretch`
- `letterSpacing`
- `textDecorationLine`
- `lineHeight`
- `textUnderlineOffset`
- `paddingHorizontal`
- `paddingVertical`
- `style-border`
- `textDecorationStyle`
- `thickness-border`
- `textDecorationThickness`
- `transform`

You must combine these property names with `Text` and the variant's name.
For example, you can set a light green background for the `abbr` variant and add some horizontal padding
with the following theme variables:

```json copy
"color-bg-Text-abbr": "lightgreen",
"paddingHorizontal-Text-abbr": "8px"
```
