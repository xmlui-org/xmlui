%-STYLE-START

The `Text` component uses these theme variables:

- `align-vertical`
- `border-radius`
- `color`
- `color-bg`
- `color-border`
- `color-decoration`
- `font-family`
- `font-size`
- `font-weight`
- `font-style`
- `font-stretch`
- `letter-spacing`
- `line-decoration`
- `line-height`
- `offset-decoration`
- `padding-horizontal`
- `padding-vertical`
- `style-border`
- `style-decoration`
- `thickness-border`
- `thickness-decoration`
- `transform`

You must combine these property names with `Text` and the variant's name.
For example, you can set a light green background for the `abbr` variant and add some horizontal padding
with the following theme variables:

```json copy
"color-bg-Text-abbr": "lightgreen",
"padding-horizontal-Text-abbr": "8px"
```
