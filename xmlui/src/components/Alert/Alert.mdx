# Alert

%-PROP-START statusColor

The value of this optional property sets the string to provide a color scheme for the Alert.

| Value       | Description                                        |
| :---------- | :------------------------------------------------- |
| `primary`   | Primary theme color **(default)**, no default icon |
| `secondary` | Secondary theme color, no default icon             |
| `success`   | Success theme color, "success" icon                |
| `warning`   | Warning theme color, "warning" icon                |
| `danger`    | Danger theme color, "danger" icon                  |
| `info`      | Info theme color, "info" icon                      |
| `light`     | Light theme color, no default icon                 |
| `dark`      | Dark theme color, no default icon                  |

%-PROP-END
