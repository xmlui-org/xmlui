## Using Link

### `Link` Appearance

You can use the `label` and `icon` properties of a `Link` to set its text and icon to display. If you want a custom appearance, you can nest your visual representation into `Link`:

```ueml copy {6-14}
<Link to="https://ncrm.azurewebsites.net/" target="_blank">
  <HStack gap="0.5rem" verticalAlignment="center">
    <Stack width="16px" height="16px" backgroundColor="purple" />
    XMLUI introduction
  </HStack>
</Link>
```

%-PROP-START target

| Value     | Description                                                   |
| --------- | ------------------------------------------------------------- |
| `_blank`  | Opens the link in a new window or tab                         |
| `_self`   | Opens the link in the same frame as it was clicked (default)  |
| `_parent` | Opens the link in the parent frame                            |
| `_top`    | Opens the link in the full body of the current browser window |

The following sample opens its link in a new tab:

<Playground
  previewOnly={true}
  name="Example: target"
  height={40}
  app={`
    <Link to="https://ncrm.azurewebsites.net/" target="_blank">
      Open XMLUI overview in a new tab
    </Link>
  `}
/>

%-PROP-END

%-EVENT-START click

This event is fired when the user clicks the link. With an event handler, you can define how to respond to the user's click.

The following sample displays a message when the user navigates to the link:

```ueml
<Link label="Check my status"
  onClick="window.alert('You are on the Status page.');" />
```

<Playground
  previewOnly={true}
  name="Example: navigate"
  height={40}
  app={`
    <Link label="Check my status" 
      onClick="window.alert('You are on the Status page.');" />
  `}
/>

%-EVENT-END

%-STYLE-START

The `Link` component uses these theme variables to customize its appearance:

- `color-decoration`
- `color-text`
- `font-weight`
- `line-decoration`
- `offset-decoration`
- `style-decoration`
- `thickness-decoration`

The following sample theme sets a few visual properties of `Link`:

```json
{
  "id": "custom",
  "name": "Custom Theme",
  "themeVars": {
    "color-text-Link--hover": "orangered",
    "color-decoration-Link": "purple",
    "thickness-decoration-Link": "4px",
    "style-decoration-Link": "dotted"
  }
}
```

<Playground

        "style-decoration-Link": "dotted",  
      },
    },
  ]}
  app={`
    <Link>I have a fancy style when you hover me!</Link>
