## Using `SpaceFiller`

In a `Stack`, `SpaceFiller` pushes the children following it to the other end of the container:

```ueml copy {3}
<HStack>
  <Stack width="36" height="36" backgroundColor="red" />
  <SpaceFiller />
  <Stack width="36" height="36" backgroundColor="blue" />
</HStack>
```

<Playground
  name="SpaceFiller in an HStack"
  height={80}
  app={`

    <SpaceFiller />

  </HStack>
  `}
/>

In a `FlowLayout`, `SpaceFiller` acts as a line break for a row. The children following the `SpaceFiller` enters a new line.

```ueml copy {3}
<FlowLayout>
  <Stack width="20%" height="36" backgroundColor="red" />
  <SpaceFiller />
  <Stack width="20%" height="36" backgroundColor="green" />
  <Stack width="20%" height="36" backgroundColor="blue" />
</FlowLayout>
```
