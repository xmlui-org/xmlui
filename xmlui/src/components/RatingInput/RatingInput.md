%-DESC-START

`RatingInput` is a star-based rating input control. Users can select a value from 1 up to `maxRating` (default 5) by clicking on the stars.

**Key features:**
- **Star rating**: Click stars to set a value from 1 to max (default 5, max 10)
- **Placeholder**: Optional placeholder when value is empty
- **Validation**: Supports validation status and invalid messages like other inputs
- **APIs**: `value`, `setValue(value)`, `focus()`

%-DESC-END
