%-PROP-START value

The following example demonstrates using it:

```ueml copy {2-6}
<VStack padding="1rem" gap="1rem">
  <ProgressBar value="0"/>
  <ProgressBar value="0.2"/>
  <ProgressBar value="0.6"/>
  <ProgressBar value="1"/>
  <ProgressBar value="1.2"/>
</VStack>
```

%-STYLE-START

The `ProgressBar` component uses these theme variables to customize its appearance:

- `color-bg-ProgressBar`
- `color-indicator-ProgressBar`
- `radius-ProgressBar`
- `radius-indicator-ProgressBar`
- `thicness-ProgressBar`

This sample shows how you can style `ProgressBar`:

```ueml copy
<Theme
  color-bg-ProgressBar="cyan"
  color-indicator-ProgressBar="purple"
  thickness-ProgressBar="12px"
  radius-indicator-ProgressBar="12px"
  radius-Progressbar="4px">
  <VStack padding="1rem" gap="1rem">
    <ProgressBar value="0"/>
    <ProgressBar value="0.2"/>
    <ProgressBar value="0.6"/>
    <ProgressBar value="1"/>
    <ProgressBar value="1.2"/>
  </VStack>
</Theme>
```
