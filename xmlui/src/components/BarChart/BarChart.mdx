%-PROP-START layout

This property determines the orientation of the bar chart.
The `vertical` variant specifies the horizontal axis as the primary and lays out the bars from left to right.
The `horizontal` variant specifies the vertical axis as the primary and has the bars spread from top to bottom.

```ueml copy {2}
<BarChart
  layout="horizontal"
  data="{[
    {
      "stage": "Prospecting",
      "value": 231
    },
    {
      "stage": "Qualification",
      "value": 56
    },
    {
      "stage": "Proposal",
      "value": 73
    },
    {
      "stage": "Negotiation",
      "value": 20
    },
    {
      "stage": "Closed Won",
      "value": 5
    }
  ]}"
/>
```
