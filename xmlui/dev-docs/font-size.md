# Components with Font-Size Specifications

## Components with Theme Variables (fontSize-*)

### Text & Display Components
- **Text**: `fontSize-Text-{variant}` (keyboard, sample, sup, sub, title, subtitle, placeholder, paragraph, subheading, tableheading, secondary)

### Markdown Component
- **Markdown**: `fontSize-H1-markdown`, `fontSize-HtmlTable`, `fontSize-HtmlThead`, `fontSize-HtmlTh`, `fontSize-HtmlTr`, `fontSize-HtmlTd`, `fontSize-Text-markdown`
